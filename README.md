# jsnapoli1 Claude Code Plugins

A personal collection of Claude Code plugins.

## Installation

Add this marketplace to Claude Code:

```
/plugin marketplace add github:jsnapoli1/claude-code-plugins
```

Then install any plugin:

```
/plugin install chat-gists
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [chat-gists](./chat-gists) | Export Claude Code chat transcripts to GitHub Gists |
